Getting Started

What is this system?

This mentorship management system helps you:

  • Add and manage mentors and mentees
  • Schedule tutoring sessions
  • Track progress and completion
  • View everything in a user-friendly calendar
Your First Steps
  1. Add Mentors: Go to "Mentors" → "Add New Mentor" to add tutors
  2. Add Mentees: Go to "Mentees" → "Add New Mentee" to add students
  3. Assign Mentors: Use "Mentees" → "Assign Mentor" to pair them up
  4. Schedule Sessions: Use "Sessions" → "Schedule Session" to book tutoring times
  5. Track Progress: Check the Dashboard and Calendar to monitor everything

Managing Mentors (Tutors)

Adding a New Mentor
  1. Click on "Mentors" in the top navigation menu
  2. Click "Add New Mentor"
  3. Fill in the required information:
    • Name: The mentor's full name
    • Roll Call: A unique identifier (like student ID)
    • Subjects: What subjects they can teach (separate with commas)
    • Max Students: How many students they can handle at once (default is 5)
  4. Click "Add Mentor" to save
Important: Each mentor can only conduct 7 total sessions before they retire from the system.
Viewing Mentor Information
  • Go to "Mentors" → "View All Mentors"
  • Click on any mentor's name to see detailed information
  • You'll see their assigned students, completed sessions, and remaining capacity

Managing Mentees (Students)

Adding a New Mentee
  1. Click on "Mentees" in the navigation menu
  2. Click "Add New Mentee"
  3. Fill in the information:
    • Name: The student's full name
    • Roll Call: A unique identifier
    • Subject: What subject they need help with
    • Lessons Remaining: How many tutoring sessions they need
  4. Click "Add Mentee" to save
Assigning Mentors to Mentees
  1. Go to "Mentees" → "Assign Mentor"
  2. Select a student who doesn't have a mentor yet
  3. Choose an available mentor who teaches that subject
  4. Click "Assign Mentor"
Note: The system will only show mentors who have space for more students and teach the right subject.

Scheduling Sessions

How to Schedule a Session
  1. Go to "Sessions" → "Schedule Session"
  2. Select the mentor and mentee for the session
  3. Choose a date and time
  4. Optionally add notes about what to cover
  5. Click "Schedule Session"
Tip: You can also quickly schedule sessions from the calendar by clicking on any date!
Managing Session Status

Sessions can have different statuses:

  • Scheduled - Session is planned but hasn't happened yet
  • Completed - Session finished successfully
  • Cancelled - Session was cancelled
  • Missed - Someone didn't show up
  • Rescheduled - Session moved to a different time

You can update these statuses from the calendar view by clicking on any session.

Using the Calendar View

Reading the Calendar
  • Blue events: Scheduled sessions
  • Green events: Completed sessions
  • Red events: Cancelled or missed sessions
  • Gray events: Session summaries (e.g., "Completed: 3")
Calendar Actions
  • Click on a date: Quickly schedule a new session
  • Click on an event: View details and update status
  • Use the view buttons: Switch between month, week, and day views
  • Navigation arrows: Move between different time periods
Filtering Sessions

Use the filter section to:

  • View sessions by specific date ranges
  • Filter by mentor or status
  • Find past sessions quickly

Tracking Progress

Dashboard Overview

The dashboard shows you:

  • Total number of mentors, mentees, and sessions
  • Sessions happening today
  • Quick actions for urgent tasks
  • System status and alerts
Sessions Due Today

The dashboard highlights sessions scheduled for today, allowing you to:

  • See which sessions are coming up
  • Quickly mark sessions as completed, cancelled, or missed
  • Contact participants if needed

Common Issues & Solutions

Can't Assign a Mentor

Problem: No mentors appear in the assignment dropdown

Solutions:

  • Check if the mentor teaches the subject the mentee needs
  • Verify the mentor hasn't reached their student limit
  • Make sure the mentor hasn't completed 7 sessions (retirement limit)
Can't Schedule a Session

Problem: Error when trying to schedule

Solutions:

  • Make sure the mentee is assigned to the selected mentor
  • Check that both participants are available
  • Verify the date and time are in the future
  • Ensure the mentee has remaining lessons
Data Not Showing

Problem: Information seems missing or outdated

Solutions:

  • Refresh the page (F5 or Ctrl+R)
  • Check if you're viewing the correct date range
  • Clear any active filters
  • Make sure data was saved properly (look for success messages)

Tips & Best Practices

Getting the Most From the System
  • Regular Updates: Mark session statuses promptly to keep accurate records
  • Use Notes: Add session notes to track what was covered
  • Check the Dashboard: Start each day by reviewing today's sessions
  • Plan Ahead: Schedule sessions in advance to avoid last-minute conflicts
  • Monitor Limits: Keep an eye on mentor session counts and mentee lesson remaining
Data Safety
  • Double-check information before saving
  • Use clear, consistent naming for roll calls
  • Regularly review assignments to ensure they're still appropriate
  • Look for success/error messages after each action
Efficiency Tips
  • Keyboard Shortcuts: Use Tab to navigate between form fields quickly
  • Quick Schedule: Click directly on calendar dates for faster scheduling
  • Bulk Actions: Use the bulk assignment feature for multiple mentees
  • Filters: Save time by using calendar filters to find specific sessions
